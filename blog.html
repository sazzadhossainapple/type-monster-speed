What is the difference between local storage and session storage?

  1. The Local storage capacity of local storage is 5MB/10MB. But the Session storage capacity of session storage is 5MB.
  2. Their main difference is that local storage data does not expire, whereas session storage data gets cleared when the user closes its browser window.
  3. In local and Session storage there is no transfer of data to the server.
